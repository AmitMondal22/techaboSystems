Overview

TSGT W100 is a WiFi IoT gateway that can interconnect WiFi, Modbus RS485, Modbus TCP, IO and provides APIs (REST API + MQTT) cloud services. It can be applied to the new energy vehicle system of Internet of Things + battery + vehicle, BMS, smart charging pile, industrial automation, smart home, intelligent building, smart agriculture, energy management and other fields. Built-in 4G Modem.

Special Features

Smart edge IoT gateway; can connect up to 100+ IOT (20x less cost than competitive IoT gw) ; Rich API (REST + MQTT) for robust integration
